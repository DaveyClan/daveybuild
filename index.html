Davey Clan Kodi Repository

This repository contains the Davey Clan Wizard and repository addon for Kodi.

Addons

  • Davey Clan Repository
    Kodi repository addon to install the Davey Clan Wizard.
    Download Repository ZIP
  • Davey Clan Wizard
    Installs the Davey Clan Kodi build.
    Download Wizard ZIP

Follow the instructions in Kodi to install the repository addon first, then install the wizard.
